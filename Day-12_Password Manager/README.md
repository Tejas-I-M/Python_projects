# Day 12 - Password Manager

## Concepts Used

* Encapsulation
* Abstraction
* File Handling (JSON)
* Encoding (Base64)

## Features

* Store account passwords
* Retrieve passwords
* Persistent storage using JSON
* Basic encoding for safety

## Description

This project is a simple password manager that allows users to store and retrieve passwords for different services. Passwords are encoded before storage to avoid plain text exposure.

## Learning Outcome

* Learned secure data handling basics
* Practiced file-based storage using JSON
* Understood encoding vs encryption
* Built a real-world useful application
