# Day 14 - Simple Chatbot

## Concepts Used

* Encapsulation
* Abstraction
* Conditional Logic
* User Interaction

## Features

* Takes user input
* Responds based on predefined rules
* Continuous interaction loop
* Exit command support

## Description

This project is a rule-based chatbot that simulates conversation using simple keyword matching. It demonstrates how basic conversational systems are built using conditional logic.

## Learning Outcome

* Learned how chat systems process user input
* Practiced building interactive applications
* Understood rule-based decision systems
* Built a simple conversational AI foundation
