# Day 10 - E-Commerce Order System

## Concepts Used

* Aggregation
* Encapsulation
* Abstraction
* Object Interaction

## Features

* Create products and customer
* Add products with quantity to order
* Calculate total cost
* Display order summary

## Description

This program simulates a simple e-commerce ordering system. It models real-world entities such as Product, Customer, and Order using object-oriented programming.

The system allows adding multiple products to an order and calculates the total cost dynamically.

## Learning Outcome

* Learned how to design a multi-class system
* Understood aggregation through Order–Product relationship
* Practiced object interaction in a real-world scenario
* Applied abstraction to simplify user interaction
