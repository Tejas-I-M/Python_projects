# Day 7 - ATM Machine System

## Concepts Used

* Encapsulation
* Data Hiding
* Abstraction
* State Management

## Features

* PIN authentication
* Deposit and withdrawal operations
* Balance checking
* Menu-driven interaction
* Input validation for transactions

## Description

This program simulates a basic ATM system where a user authenticates using a PIN and performs banking operations such as deposit, withdrawal, and balance inquiry.

Sensitive data like balance and PIN are protected using encapsulation, ensuring controlled access through methods.

## Learning Outcome

* Learned how to secure data using encapsulation
* Understood how to manage object state dynamically
* Practiced building interactive, menu-driven systems
* Applied abstraction to hide internal implementation
