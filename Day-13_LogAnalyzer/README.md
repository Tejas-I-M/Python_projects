# Day 13 - Log Analyzer

## Concepts Used

* Encapsulation
* Abstraction
* File Handling
* Data Processing

## Features

* Read log file
* Count INFO, ERROR, WARNING logs
* Extract error messages
* Display summary

## Description

This program analyzes log files and extracts useful information such as error counts and log distribution. It simulates real-world log monitoring used in backend systems.

## Learning Outcome

* Learned log parsing techniques
* Practiced file handling
* Built a system for analyzing real-world data
* Improved data processing skills
