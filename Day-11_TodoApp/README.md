# Day 11 - CLI To-Do Application

## Concepts Used

* Encapsulation
* Abstraction
* File Handling (JSON)
* State Persistence

## Features

* Add tasks
* View tasks
* Mark tasks as complete
* Delete tasks
* Persistent storage using JSON

## Description

This project is a command-line To-Do application that allows users to manage tasks efficiently. Tasks are stored in a JSON file, ensuring persistence across program runs.

The system is designed using object-oriented principles to separate business logic from user interaction.

## Learning Outcome

* Learned how to build a real-world CLI application
* Practiced file handling with JSON
* Understood how to persist application state
* Applied clean OOP design in a practical project
