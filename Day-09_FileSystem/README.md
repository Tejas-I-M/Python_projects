# Day 9 - File Management System

## Concepts Used

* Encapsulation
* Abstraction
* File Handling
* State Management

## Features

* Create files with content
* Read file content
* Delete files
* List all files in directory
* Menu-driven interface

## Description

This program simulates a basic file management system using Python and object-oriented design.

All file operations are encapsulated within a class, providing a clean and controlled interface for interacting with the file system.

## Learning Outcome

* Learned how to integrate file handling with OOP
* Practiced abstraction by hiding file operations
* Built a menu-driven system for real-world interaction
* Understood how to manage persistent state using files
