# Day 8 - Hotel Booking System

## Concepts Used

* Aggregation
* Encapsulation
* State Management
* Object Interaction

## Features

* Add rooms to hotel
* View available rooms
* Book a room
* Checkout a room
* Track room availability

## Description

This program simulates a hotel booking system using object-oriented design. The system manages multiple rooms and allows booking and checkout operations.

Each room maintains its own state (booked/available), while the Hotel class manages and interacts with multiple Room objects.

## Learning Outcome

* Learned how multiple classes interact in a real-world system
* Understood aggregation through Hotel–Room relationship
* Practiced managing dynamic object states
* Built a system with realistic booking logic
